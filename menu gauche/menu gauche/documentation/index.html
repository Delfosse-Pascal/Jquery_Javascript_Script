Sidebar

This Project has following folder structure :

  • index.htmlthe main file, containes html structure and jQuery effects
  • css/* (containes two css file )
  • css/basic.css sidebar design, the basic the design not including colors.
  • css/font-awesome.css icon font, facebook, twitter, instagrm .. a lot more.
  • css/skins/* contains 6 diffrent colors.
  • fonts/* icon font, by http://fontawesome.io , free font.
  • imgs/*contains the logo image.

How to change skins :

to change the default skin, you can easly choose one of the available skins. Follow this steps :

  1. Open the index.html file
  2. Go to line 7, or search for : 
     <link rel="stylesheet" type="text/css" href="assets/css/skins/pink.css">
  3. Replace the word pink with whatever skin you want: 
    			#  dark
			#  darkblue
			#  green
			#  lightblue
			#  orange
			#  pink
